Test Information

Each set included in this library is tested two ways:

  1. By applying the generic font family and icon styles to unicode characters that exist directly in the HTML:

    i { @extend %ss-symbolicons-line; }

  2. By applying the %ss-ICON via pseudo elements to custom selectors.

      .action:before { @extend %ss-action; }
  .activity:after { @extend %ss-activity; }
  …

I don't see the fonts!

Well, of course you don't. You either: (a) don't have the font installed on your system, or (b) don't have the font installed in the directory the test css is looking for: ../fonts/ss-symbolicons-line/webfonts. That being the case, you've probably got some 404 errors in your console. Anyway, this doesn't matter. It works if you follow the usage instructions.
